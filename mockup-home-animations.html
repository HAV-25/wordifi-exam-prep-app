CHANGE 2 — GLOW BORDER

Travelling glow on CTA bar

CHANGE 3 — RING + SCORE

Animated readiness ring

On screen mount, the green ring fills from 0% to the user's score (e.g. 93%) over ~1.2s using an ease-out curve.

The percentage number simultaneously counts up from 0 → 93 at the same speed, giving a "powering up" feel.

Both animations use the existing motion.duration.scoreRing constant from theme/motion.ts.
